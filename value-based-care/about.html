About this project

A short academic site exploring the principles and practice of value-based care.

Design goals

This site is organised to meet the academic assignment requirements: multi-page structure, semantic markup, external stylesheet, CSS Grid layout, Flexbox use, multi-column text where appropriate, and an accessible structure. Images should have meaningful alt text and headings are ordered for screen readers.

Navigation uses a simple, keyboard-focusable list. Layout relies on CSS Grid for the main structure and Flexbox for the header and footer alignment. The textual section uses CSS multi-column flow to meet the multi-column requirement for each page.

Accessibility notes

  • All images must include alt text (replace placeholder filenames with meaningful descriptions).
  • Landmark roles (banner, main, navigation, contentinfo) are included for assistive tech.
  • Colour contrast should be tested with browser tools; change variables in `css/styles.css` as needed.
