About This Project

Understanding how healthcare can shift from volume to value for more equitable, sustainable care.

Purpose

The purpose of this website is to introduce the concept of value-based care — an approach that prioritises patient outcomes over service volume. It serves as an accessible educational platform for anyone interested in how healthcare systems transition from fee-for-service toward models that reward quality, equity, and efficiency.

In creating this project I wanted to explore how digital communication can clarify complex healthcare ideas for a broad audience. The site’s structure allows users to move from introductory explanations to policy considerations and case studies that demonstrate the impact of value-driven models on patients and providers.

Design approach

The design emphasizes clarity, accessibility, and calm. A minimal palette, readable typography, and balanced white space foreground the content — reflecting the transparency that value-based care aims for. Layout uses CSS Grid and Flexbox for consistent structure and responsive alignment.

Images are responsive and accompanied by descriptive alt text. Semantic HTML and clear headings improve screen-reader navigation. The navigation and page layout create a smooth, uninterrupted user journey, emphasizing simplicity and clarity.
